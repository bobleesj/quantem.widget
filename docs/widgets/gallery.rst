Widget Gallery
==============

Visual reference for every quantem.widget state in light and dark themes.

.. tip::

   Run ``python scripts/generate_catalog.py --docs`` to regenerate these screenshots
   after UI changes. Without ``--docs``, screenshots go to ``tests/screenshots/catalog/``
   only (for local QA).
